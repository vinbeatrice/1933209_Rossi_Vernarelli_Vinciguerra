# SYSTEM DESCRIPTION
A distributed automation platform capable of ingesting heterogeneous
sensor data, normalizing it into a unified internal representation, evaluating simple automation
rules, and providing a real-time dashboard for habitat monitoring.

# USER STORIES
1) As a Mars operator, I want to visualize current data coming from REST sensors so that I can monitor the situation.
2) As a Mars operator, I want to visualize telemetries coming from sensors so that I can monitor the situation.
3) As a Mars operator, I want to see plots of the recent history of data from all the sensors so that I can visualize its trend.
4) As a Mars operator, I want to be able to select the specific sensor data I want to plot so that I can analyze them in detail.
5) As a Mars operator, I want to be able to manually refresh values captured by the sensors so that I can get the most up to date information.
6) As a Mars operator, I want to set the value of the refreshing interval (intervallo di aggiornamento del valore) so that I can decide the frequency.
7) As a Mars operator, I want to add an automation rule to automate my reaction to the environment.
8) As a Mars operator, I want to update an automation rule so that I can customize my rules.
9) As a Mars operator, I want to delete an automation rule so that I can get rid of it if I don’t need it anymore.
10) As a Mars operator, I want to be able to visualize the actual state of all the actuators so that I can monitor the situation.
11) As a Mars operator, I want to manually change the state of the actuators so that I can have more control.
12) As a Mars operator, I want the values over the threshold to be highlighted so that I can easily understand what is happening.
13) As a Mars operator, I want to see some metrics (average, max, min) about received data so that I can analyse them.
14) As a Mars operator, I want to receive a notification when an automation rule is triggered so that I know when the system reacts automatically. (log)
15) As a Mars operator, I want to be able to export useful data into pdf files so that I can process them offline.
16) As a Mars operator, I want to be able to be able to activate/disable an automation rule so that I can have more flexibility and handle critic situations.
